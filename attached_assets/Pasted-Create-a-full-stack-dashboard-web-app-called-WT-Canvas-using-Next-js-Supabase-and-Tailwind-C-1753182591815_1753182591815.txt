## 📄 Content Management
Each Website Content section includes:
- A table showing:
  - Title
  - Status (`idea`, `draft`, `review`, `approved`, `scheduled`, `live`)
  - Approval status (`pending`, `approved`, `rejected`)
  - Scheduled publish date
  - Published status
  - Last updated
- Action buttons:
  - Edit
  - Publish Now
  - Republish

### Edit Page:
- Full form to edit metadata and content blocks
- Rich content block builder (`heading`, `text`, `image`, `quote`, `list`, `cta`, etc.)
- Buttons to trigger OpenAI (Well Told custom GPT) for:
  - Title generation
  - Summary writing
  - Section drafting or improvement
- Save/update to Supabase

## 🚀 Publishing Workflow
- “Publish Now” and “Republish” trigger a POST to the correct **Framer CMS subdomain** based on type (`blog`, `lead_magnet`, or `campaign_page`)
- Store `published_at` timestamp on publish
- Add logic to **auto-publish** approved entries:
  - Every 5 minutes, check for:
    - `status = 'approved'`
    - `published_at IS NULL`
    - `scheduled_publish_date < now()`
  - Automatically publish those items
